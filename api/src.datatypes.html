src.datatypes.cell module

cell.py

Defines the Cell dataclass, which represents a single segmented cell region extracted from a segmentation mask.

class src.datatypes.cell.Cell(id, points=<factory>, clipping=False)[source]

Bases: object

Represent a single cell region within a segmentation mask.

A Cell groups all Point instances that fall within its boundary, along with metadata describing the cell’s identity and whether it lies on the edge of the sampled area.

Variables:

  • id (int) – The cell’s identifier. Corresponds to the integer label assigned to this cell in the segmentation mask.

  • points (list[Point]) – The points located within the cell’s area. Defaults to an empty list if none are provided.

  • clipping (bool) – Whether the cell is clipped (cut off) by the border of the sample area. Defaults to False.

src.datatypes.sample module

sample.py

Defines the Sample dataclass, which bundles together all the data required to process a single sample: its metadata, detected cell regions, manually placed points, segmentation mask, sample area, and resulting evaluation scores.

class src.datatypes.sample.Sample(metadata, cells, points, mask, sample_area, original_image=None, results=<factory>)[source]

Bases: object

Contain all the data needed to process one sample.

A Sample ties together a sample’s Metadata, its detected Cell regions, the manually placed Point instances used for ground truth comparison, the labeled segmentation mask to be assessed, the bounds of the sampled area, and the resulting evaluation Results.

Variables:

  • metadata (Metadata) – Information describing the current sample.

  • cells (dict[int, Cell]) – A mapping of cell ID to the corresponding Cell region for every cell detected in the sample.

  • points (list[Point]) – The list of manually placed points within the sample.

  • mask (numpy.typing.NDArray[numpy.uint16]) – The labeled cell segmentation mask to assess, where each cell region is identified by its integer label.

  • sample_area (SampleArea) – The area of the sample used to determine which cells and points are in bounds.

  • original_image (numpy.typing.NDArray[numpy.uint16], optional) – The original image the mask was generated from. Optional; defaults to None if not available.

  • results (Results) – The results of processing this sample, including true/false positive counts and precision, recall, and F1 scores. Defaults to an empty Results instance.

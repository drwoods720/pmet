src.outputs.output module

output.py

Defines the Output abstract class, which establishes the interface that all output generators must implement to turn processed sample data into a finished output artifact, such as a CSV report or a visualization, and write it to disk.

class src.outputs.output.Output[source]

Bases: ABC

Abstract base class for output generators.

Subclasses of Output are responsible for generating the resulting output artifact from processed data. Each concrete subclass must implement run().

src.outputs.overlay module

overlay.py

Defines the Overlay output generator, which produces a colour-coded overlay image visualizing per-cell accuracy against the original segmentation mask.

overlay_image(background, overlay, alpha=1.0)[source]

Overlay one image on top of another with optional transparency.

Parameters:

  • background (npt.NDArray[np.uint16]) – The background image.

  • overlay (GenericAlias[uint16]) – The image to overlay on top of the background image.

  • alpha (float) – The transparency value of the overlay image, in the range 0 (fully transparent) to 1 (fully opaque).

Returns:

The background image with the overlay image blended on top of it.

Return type:

GenericAlias[uint16]

run(data, output_directory)[source]

Generate and write the accuracy overlay image for a sample.

Builds the accuracy-coded image via generate_accuracy_mask(), optionally blends it over the sample’s original image via overlay_image(), and writes the result to output_directory.

Parameters:

  • data (Sample) – The processed sample data to generate the overlay image from.

  • output_directory (Path) – The directory to write the generated overlay image to.

Returns:

None. The generated image will be written as a side effect to a file within output_directory.

Return type:

None
